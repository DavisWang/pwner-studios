# Game Directory (pwner studios)

## Overview

- Pwner studios is a one man game studio whose goal is to recreate nostalgic retro style games from the early 2000s and share that nostalgic joy of the early 2000s with everyone. This project is to establish a web presence (via github pages) that serves as the portfolio page for pwner studio’s works.  
- Games under pwner studio’s portfolio:  
  - Burn the Village  
    - https://github.com/DavisWang/burn-the-village  
    - [https://daviswang.github.io/burn-the-village/](https://daviswang.github.io/burn-the-village/)  
  - Dots  
    - https://github.com/DavisWang/dots  
    - [https://daviswang.github.io/](https://daviswang.github.io/burn-the-village/)dots/  
  - Age of War  
    - https://github.com/DavisWang/aow  
    - [https://daviswang.github.io/aow/](https://daviswang.github.io/burn-the-village/)  
  - RPS Arena  
    - [https://github.com/DavisWang/rps-arena](https://github.com/DavisWang/rps-arena)  
    - [https://daviswang.github.io/](https://daviswang.github.io/burn-the-village/)rps-arena  
  - And more to come  
- Design a sleek, highly interactive but simple landing page to showcase the above works. With affordances given to the user to learn more about each of the above projects in more detail, and with links to follow if they want to learn even more.   
- The page should minimize the use of text. Short sentences are fine at the top level. Up to 3-4 sentences are fine for the project detail view. Use assets from respective projects to make the page more interesting.  
- Color Palette: The color palette should be primarily dark, retro, and 8bit. It should use colors, interactions, visual elements that invoke the early days of computer gaming in the 90s and early 2000s. Ideally, just the color palette should give the audience a sense of what pwner studios is about.  
- Finally, link to [https://daviswang.github.io/](https://daviswang.github.io/burn-the-village/)personal-website/ as the man behind the studio.
